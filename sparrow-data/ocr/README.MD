# Sparrow Data OCR

## Description

This module implements OCR functionality for Sparrow ML.

## Install

1. Install

```
pip install -r requirements.txt
```

## Plain OCR

If you want to test how OCR works, you can run scripts

1. PaddleOCR

```
python ocr_paddleocr.py
```
